Use this agent when you want help with:

- Authentication, JWT, authorization, and user flow logic.
- Frontend React/Vite UI and API integration in `frontend/src`.
- Debugging project-specific errors, enhancing endpoints, or updating API docs.

Prefer answers that:
- reference this repository's files and current structure,
- avoid generic unrelated frameworks or libraries,
- provide concrete code edits or fix suggestions,
- keep changes scoped to the project.
